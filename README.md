# UpdatedPortfolio

This is a modern 3D portfolio template built with **HTML, CSS, and Three.js**, tailored for a 2nd-year BSc Applied Mathematics and Computer Science (UJ) student aspiring to become a Machine Learning Engineer. 

## Features
- Interactive 3D background and project highlights using Three.js
- Sections: About, Projects, Certifications, Skills, Contact
- Responsive design for desktop and mobile
- Light/Dark mode toggle
- Smooth scrolling and hover effects
- Includes APM and Computer Science courses, certifications, and 3 projects
